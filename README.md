This repository contains images from a prototype version of Magic Missile Wizard Duel (MMWD) along with diagrams and the implementation report for MMWD. MMDWD which was built by Ryan Vincoy, Roger Chiu, Hunter Olson, and Seth Traman for the Software Engineering course at UIC. Magic Missile Wizard Duel was designed by Annie You, Emily Mendoza, Kevin Cordero, and Nandana Sheri.
